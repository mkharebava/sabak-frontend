# Phishing Simulation — Frontend

საბაკალავრო კვლევის პროექტი: ვიშინგის სიმულაცია და მომხმარებელთა ცნობიერება

## პროექტის შესახებ

ეს პროექტი წარმოადგენს ვიშინგ სიმულაციის პლატფორმის frontend ნაწილს, რომელიც შეიქმნა საბაკალავრო კვლევის ვის ფარგლებში. პლატფორმა სიმულირებს რეალური ვიშინგ შეტევების სცენარებს საგანმანათლებლო და კვლევითი მიზნებისთვის.

## Live Demo

- **პლატფორმა:** https://techop.vercel.app
- **Admin Panel:** https://techop.vercel.app/admin
